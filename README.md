# BonDriver_BDA
##凡銅鑼の野望 ～Genericの章～

BonDriver_BDA.iniでDefaultNetwork="4KBS/CS"と記述すると、ISDB-S3に対応します。

[Tuner]

UseSpecial="TBS" ;ここでTBS.dll読み込む

Guid=

FriendlyName0=

Name="ISDB-S"

DVBSystemType=1

DefaultNetwork="4KBS/CS"

SignalLevelCalcType="SSStrength"

StrengthCoefficient=1000.0

[TBS]

SetTSID=1

LNBPowerON=1
